# decodelabs-project-4
This Python program is a simple General Knowledge Quiz Application that asks users three multiple-topic questions, evaluates their answers, and displays a final score along with performance feedback. The program demonstrates fundamental Python concepts such as variables, user input, conditional statements, string manipulation, and score tracking.

# General Knowledge Quiz Application

A simple command-line based General Knowledge Quiz built using Python. This project allows users to answer multiple-choice style questions, checks their responses, calculates the score, and displays the final result with feedback.

# Features

- Asks multiple general knowledge questions
- Validates user answers
- Calculates and displays the final score
- Provides instant feedback for each question
- Displays performance-based messages
- Beginner-friendly implementation

# Technologies Used

- Python

# How It Works

The program presents a series of general knowledge questions to the user.

Users can:

- Enter answers to each question
- Receive immediate feedback on whether their answer is correct or incorrect
- View their final score after completing the quiz
- Receive a performance message based on their score

The program uses score tracking and conditional statements to evaluate user responses.

# How to Run

1. Install Python on your system.
2. Save the code in a file named `quiz.py`
3. Open Terminal or Command Prompt.
4. Run the program using:

```bash
python quiz.py
```
